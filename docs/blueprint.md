# **App Name**: CampusConnect

## Core Features:

- Secure Authentication: Email/Password and optional OAuth authentication with role-based access control (Admin, Staff, Teacher, Student, Parent).
- School Profile Management: Set up and manage school profile, academic year, classes, grades, sections, and subjects.
- Student Management: Student admission, registration, profiles, class assignment, promotion, and ID generation.
- Teacher & Staff Management: Teacher profiles, subject assignment, staff roles, attendance tracking, and basic salary management.
- Attendance Tracking: Daily attendance for students, teachers, and staff with attendance reports.
- Notifications and Announcements: In-app notifications, announcements, and parent-teacher communication.
- AI-Powered Report Summarization: AI tool that can generate concise summaries of student performance reports and attendance reports.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey trust and stability.
- Background color: Light gray (#F4F6F6), creating a clean and modern look.
- Accent color: Teal (#008080) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral, and readable experience.
- Consistent and clear icons representing different modules and actions within the system.
- Clean and intuitive layout with clear navigation for different user roles.
- Subtle transitions and animations to enhance user experience.
